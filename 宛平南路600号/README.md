# 说书人手册

## 设计思路

癫狂，娱乐，强大
判定难度不高，便于玩家理解
适合很菜的玩家和老手混玩，不太需要盘逻辑
节目效果拉满
没有中毒干扰
善良的能力都比较强大并显然，即使新手拿到也能有事可做，但老手拿到也不会立刻摧毁游戏。
爪牙均为暴露型且存在伪装空间，恶魔围绕平安夜和外来者数量逻辑，哈迪和朋友游戏以节目效果为主
外来者均为软调整型，善良不提供外来者调整逻辑

## 角色取舍

互动复杂，难以理解，相克过多而舍弃的角色：
召唤师/小怪宝/天才/预兆/牙噶巴卜

与哈迪有所冲突的角色：公主、半兽人、僧侣

碾骨魔: 对比方古触发频率低又更隐蔽，舍弃
酒鬼/哲学家: 体验过差，舍弃
造谣者: 放弃额外死亡逻辑，难度过高，只保留平安夜
巡山人:太弱，换成可以看更多外来的卧底记者

## 备选角色

通灵师

## 角色运作

### 卧底记者

尽量提供无法明跳的角色（落难，疯子，傻子，邪恶角色），根据寻找难度和局势需求提供如何变化、如果采访对象是镇民、可以考虑把卧底记者变成外来者、避免过强双证。

### 精神分裂患者

类似巡山人，即使初始外来者数量为1，依然可以分裂外来者，如果初始外来者数量是2，可以增加外来者也可以不增加外来者。因分裂患者中毒的炼金术士不会具有设置调整能力。

### 吃瓜群众

可以获得任意类型的信息，比如2（共情者、厨师）、小红小明（店小二），也可以是自制能力（类似失忆者）

### 钢琴师

对于邪恶伪装的钢琴师、可以视情况调整bgm

### 穿越者

类似失忆者，可以是官方已有角色，也可以是自制角色，穿越者知道自己的能力是什么，被发现的判定类似疯狂，被发现后，穿越者会拥有类似修补匠的能力。
