# 设计思路

一个充满欺骗、谎言、和问答的剧本

# 核心角色

1. 恶魔

涡流：提供第一个信息可能错误的基点，要求剧本信息位能在涡流情况下发挥作用，比如修行者、贵族、牧师
饕餮/狂戮/方古：要求恶魔需要分析场上信息
饕餮狂戮提供多刀逻辑，镇民有固定的夜晚杀人逻辑（审判官）

2. 外来者

外来者提供软设置调整，地狱弃子、畸形秀演员（贱民）、瘟疫医生
解谜大师（提供醉酒）

3. 爪牙

由于恶魔需要获得信息，以及有炼金的存在，独眼巨人、预兆、斯芬克斯

## 镇民

1. 小精灵：提供对跳逻辑，伪装逻辑，首夜信息

2. 艺术家/小说家：提供是非信息，但是小说家过于复杂，舍弃

3. 地雷逻辑：秉笔，提供白天认出逻辑，在涡流作用下发挥水平不变

4. 炼金：考虑到恶魔也需要信息，以及娱乐性不错，减少爪牙暴露属性

5. 隐修：提供不跳逻辑

6. 剧本不考虑保护逻辑
